# 🧪 Goop Dungeon
**Goop Dungeon** is a prototype 2D platformer, side-scroller, arcade game for PC, made with Unity 6. 
In the game, you play as a green goop, whose ability is to absorb and hold objects. 
Your goal is to escape the dungeon and see the outside world with other goops!

## General Description
There are many typical 2D platformer mechanics in this game, 
such as jumping on the tiles, jumping over lava blocks, dodging enemy attacks, pushing buttons to open gates and finding the keys to the doors to get to the next level

## Game Features
- **Convenient character control:** small and big jumps, fall control, acceleration
- **Automatic collection of items.** Items can be thrown through the **E** button.
- An NPC water goop who can be given an item and **receive help from it in return.**
- If the goop likes your item, it will start **following you and giving buffs.**

## Visuals
For the visual part, the project used technologies such as: 
- Tilemaps, 
- DOTween for UI animations, 
- Unity Animator, 
- Post-processing,
- 2D lights and shadows,
- Particle System,
- Shader Graphs for damage effects,
- Music and sound effects

## 💻 Platforms
**Windows**  
Goop Dungeon is optimized for PC with keyboard controls.
- **Move** - A, D and ← → buttons,
- **Jump** - Space button,
- **Throw item** - E button,
- **Game pause** - Esc button
